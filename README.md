# Skills

מאגר של Skills עבור Claude ומודלי שפה אחרים. כל Skill הוא חבילה עצמאית (קובץ `.skill` או תיקייה עם `SKILL.md`) שמספקת למודל הוראות, סקריפטים וקבצי עזר לטיפול במשימה ייעודית.

המאגר מאורגן לפי תחום או פלטפורמה (לדוגמה `apple/`, `web/`), וכל Skill כולל תיאור (`description`) שמכוון את המודל לזהות מתי הוא רלוונטי.

## רשימת Skills

### apple

- **[apple-app-store-screenshots](apple/apple-app-store-screenshots.skill)** — ממיר תמונה כלשהי לצילום מסך שעומד בדרישות של Apple App Store Connect (iPhone, iPad, Mac, Apple TV, Apple Watch, Apple Vision Pro). מפיק PNG במידות הפיקסלים המדויקות שאפל דורשת, ב-RGB ללא ערוץ אלפא, ומטפל באי-התאמה של יחס גובה-רוחב על ידי שאלת המשתמש איך להתאים את התוכן (ריפוד לבן/שחור/צבעוני, רקע מטושטש, מתיחה או חיתוך) במקום לעוות בשקט. שימושי להגשה לחנות, ל-TestFlight, או לתיקון צילומי מסך שאפל דחתה.
